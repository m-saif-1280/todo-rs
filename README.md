# todo-rs
A TUI based todo list program in Rust with Ratatui

## Features
- **Completely local:** No cloud, no online account, no nothing
- **Terminal centric:** Built for the terminal with `ratatui`
- **Efficient:** Written in rust for performance and memory safety

## Installation

### Prerequisites
* [Rust](https://rustup.rs) v1.96+

### Build from source
```sh
git clone --depth 1 --single-branch https://github.com/m-saif-1280/todo-rs
cd todo-rs
cargo build --release
./target/release/todo-rs
```
